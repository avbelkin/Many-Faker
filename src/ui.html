Replaces text placeholders with random values
Format: {replacer[[min-]max][:format]}
Format description
Examples:
{random} > 42
{random1K} > 327
{randomN599-999} > 852
{randomE1K:.2f} > 19.99
{money500K-1M} > 958,235.00
{soon7} > 21-Jul-2023
{soonDatetime365} > 15-Jul-2024 09:42:59
{zipCode} > 100500
{moneyN10K-20K} {ccy} > 15,542.25 EUR
Where replacer one of:
random, randomN, randomE, money, moneyN, moneyE (numbers),
recent, soon, recentDatetime, soonDatetime (dates),
country, state, county, city, cityName, address, secondaryAddress, zipCode, email, dbemail, iban, aminet, crds, currency, $, ccy, bic, masked, id, guid, companyName, companySuffix, party (name+suffix), client (party + country code)
Where min and max are numbers following by scale: 0, 10, 42, 1000, 1K, 100000, 100K, 42M, 100B
Where format is d3 format for numbers or d3 time format for dates. Examples: d, .2f ,.4f, %d-%b-%Y %H:%M:%S, %d-%b-%Y
